MinusPod Fork Architecture

Fork of ttlequals0/MinusPod

Self-hosted podcast ad remover with original-audio fallback and RSS status labeling

Fork Changes

  • • Original audio fallbackNEW
  • Unprocessed episodes stream original audio immediately — no 503 blocking
  • • RSS status labelingNEW
  • [Ad-Free] / [Processing] tags in episode titles visible in podcast apps
  • • HEAD request cachingNEW
  • Cache upstream RSS lookups to prevent server lockups on feed refresh

Processing Pipeline

  • • Groq Whisper: ~50s per hour of audio
  • • NVIDIA NIM: 2-pass LLM ad detection
  • • FFmpeg: cuts detected ad segments
  • • Pattern learning: gets smarter over time
  • • Total: ~12 min for a 60-min episode
  • • Cost: $0.00 (all free tiers)

Deployment

  • • Podman Quadlet on OVH bare metal
  • • 4 Gunicorn workers (32 threads)
  • • CF Tunnel: pod.ogsapps.cc
  • • SQLite at systemd-minuspod-data
  • • 5-day retention, auto-cleanup
  • • 11 podcasts, 7000+ episodes discovered
